## Aim:

## Algorithm:
### Step 1: import numpy
### Step 2: write the array

### Step 4: print the solution
## Program:
#Program to find the rank of a matrix.
#Developed by: G.Kavya
#RegisterNumber:212225220050
import numpy as np
a=np.array([[3,2,5],[1,1,2],[3,3,6]])
solution=np.linalg.matrix_rank(a)
print(solution)
